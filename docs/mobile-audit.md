# Mobile Audit

3014 has been aligned around a shared shell and design-system components. Mobile QA must continue route-by-route.

Risk areas:

- wide data tables
- drawer/modal behavior
- dense KPI grids
- action rails
- map surfaces
- report tables

Acceptance:

- no horizontal overflow
- all CTAs reachable
- touch targets at least 44px
- readable sidebar/mobile nav
- cards stack cleanly
