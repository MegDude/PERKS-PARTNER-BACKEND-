# Volume 11: Automation

## Purpose

Define background jobs, n8n workflows, triggers, retries, failure handling, and event orchestration.

## Automation Standard

Automation should be deterministic first. Use AI only when summarization, classification, or generation creates clear value.

## Required Workflow Fields

- trigger
- condition
- action
- owner
- retries
- failure path
- audit event
- notification path
