# Volume 09: Campaign Engine

## Purpose

Define campaign lifecycle, targeting, scheduling, creative, QR tracking, analytics, and reporting.

## Lifecycle

Draft -> review -> scheduled -> active -> paused -> completed -> reported -> archived.

## Core Objects

- Campaign
- Campaign target
- Campaign asset
- Offer
- Event
- QR code
- Audience
- Delivery log
- Metrics

## Rules

Campaigns must reference registry entities and partner workspaces. They should not create duplicate partner, location, or offer records.
